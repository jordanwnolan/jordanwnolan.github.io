PepperMint

PepperMint is a personal finance website with social incentives to stay under budget. Users can make their budgets public or private and the app will broadcast updates whether the user is over or under budget. People following that user can then give them fame or shame (upvote/downvote) the broadcasted budget.

To complete this project I wrote complex rails associations and SQL queries to grab transaction data across multiple tables. I also used JavaScript and jQuery for user interaction with pie charts. Clicking on a section of the pie chart will fire off an AJAX request to filter a table of transactions down to the category that the user clicked. I also had to employ multiple polymorphic models to implement the social features for public budgets and goals being broadcast, comments, and user notifications.
